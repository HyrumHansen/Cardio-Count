Overview

About Our Project

Over the course of 3 weeks we learned python and machine learning from scratch to produce an analysis of the symptoms that can determine whether a patient has heart disease.

The two datasets that we used to create the project are a heart disease dataset and a heart failure dataset.

The main question we were trying to answer while creating this project was, "What symptoms best determine whether a patient has heart disease?"

This project will benefit people who are at risk for heart failure and doctors who want to give high-risk patients prevention plans before any treatment is necessary.

Machine Learning

Support Vector Machine

About the Algorithm

A Support Vector Machine, or SVM, is a model used to solve classification and regression problems. The model attempts to separate data into classes by drawing a "separating hyperplane." Achieving perfect separability is rarely possible, so the restrictions on separation are loosened to achieve a "good enough" model.

Heart Disease Variable Importance

The most important variables in this data set were maximum heart rate achieved, exercise angina (the pain that occurs when not enough blood flow is getting to the heart), oldpeak (ST depression induced by exercise relative to rest), and the slope of the ST segment (the, typically flat, section on an ECG reading).

Heart Failure Variable Importance

The variables that held the most importance in this data set were ejection fraction (the percentage of blood leaving the heart at each contraction), the level of serum creatinine in the blood, and the amount of time between the follow up appointment.
